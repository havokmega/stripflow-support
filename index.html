Common fixes

  • Quit and relaunch the app.
  • Update to the latest version from the Mac App Store.
  • Restart your Mac if an external display is involved.
  • Check macOS privacy permissions if you use any system integrations.

What to include in a support request

  • StripFlow version (from the app's About screen)
  • macOS version
  • Steps to reproduce the issue
  • What you expected vs what happened
  • Screenshots if relevant
